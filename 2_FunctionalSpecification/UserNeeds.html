User Needs

The following user needs are the things that the system will deliver for the user itself. These user needs will make their job easier and carrying out task in an effective manner.

  • Controlling the availability and preventing overbooking of rooms.

  • Making the users work run smoother and well organized.

  • Guarantee customer satisfaction.

  • Enable the user to add new or edit existing Guest Detail.

  • Enable the user to add new or edit existing Room Detail.

  • Enable the user to add new or edit existing Booking Detail.

  • Enable the user to add new or edit existing Extra Charge Detail.

  • Enable the user to add new or edit existing Payment Detail. Â 

  • Enable the user to delete a booking.

  • Easy to navigate through Guest details, Room details, Booking details, Payment details, ExtraCharge details, Booking cancelation details and Reports. Every details are available in the Main Menu form.

  • Enabling the user to view and print of reports.

  • Generating monthly and annual income reports to see the outcome of the organisation.

  • Creating a search that enables the user to find a particular guest.

  • Grouping guests using their age, such as teens, adults and senior.

  • To provide accessibility control to data with respect to users.

  • The ability to make booking anytime.

  • To avoid manual and repetitive work.

  • Creating a User Guide that will help the user to know how to install the system, use the system and how to troubleshoot problems.

  • Disabling the Ribbon menu, navigation pane and the Shortcut Menu.
